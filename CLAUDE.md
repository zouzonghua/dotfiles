# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Commands

```sh
make              # install all packages (auto-detects macOS → desktop profile)
make install      # install CLI packages only (server profile)
make desktop      # install CLI + GUI packages (Linux explicit desktop)
make <package>    # install a single package, e.g. make tmux
make setup        # re-run post-install: git signing, SSH permissions, shell rc injection
make check        # verify required dependencies (stow, bash, ssh, git)
make uninstall    # remove all stow symlinks and generated files
```

## Architecture

This repo uses **GNU Stow** to manage symlinks. Each top-level directory is a Stow module whose internal structure mirrors `$HOME`. For example, `tmux/.config/tmux/` → `~/.config/tmux/`.

**Never edit files under `$HOME` directly** — always modify the source files inside the repo, then run `make <module>` or `make setup` to sync.

### Module map

| Module | Destination |
|--------|-------------|
| `git/` | `~/.config/git/` (config, identity files, allowed_signers) |
| `shell/` | `~/.config/shell/` (init.sh, aliases.sh, prompt.sh, history.sh) |
| `ssh/` | `~/.ssh/config`, `~/.ssh/devcontainer` |
| `tmux/` | `~/.config/tmux/` |
| `kitty/` | `~/.config/kitty/` (GUI/macOS) |
| `aerospace/` | `~/.config/aerospace/` (macOS only) |
| `vim/` | `~/.vimrc` |
| `peco/` | `~/.config/peco/` |
| `gemini/` | `~/.gemini/` |
| `codex/` | `~/.codex/` |
| `claude/` | `~/.claude/CLAUDE.md`（只链单文件，不动 `~/.claude/` 下的运行时数据）|

**AI 配置共享源**：`gemini/.gemini/GEMINI.md`、`codex/.codex/AGENTS.md`、`claude/.claude/CLAUDE.md` 都是指向 `ai-shared/preferences.md` 的相对符号链接。修改 AI 行为规则只改 `ai-shared/preferences.md`。

### `scripts/setup.sh` — what it does

- **Shell init injection**: Appends/updates a `# BEGIN DOTFILES … # END DOTFILES` block in `~/.zshrc` and `~/.bashrc` that sources `~/.config/shell/init.sh`.
- **Git signing**: Reads `~/.config/git/personal.identity` and `~/.config/git/work.identity` to generate `~/.config/git/allowed_signers` for SSH commit signing.
- **SSH permissions**: Enforces `700` on `~/.ssh/` and `600` on config files.

### Git identity switching

`git/.config/git/config` uses `includeIf "gitdir:~/work/"` and `includeIf "gitdir:~/personal/"` to load separate identity files automatically. GPG signing uses SSH keys; `allowed_signers` is generated by `make setup`.

### Profile logic (Makefile)

- Default profile is `server` (CLI only); auto-switches to `desktop` on macOS.
- `PROFILE=desktop` adds `kitty`; Darwin always adds `aerospace`.

## Git commit convention

Use **Conventional Commits** with Chinese messages. Required content: problem/requirement description, implementation approach, reproduction path (optional).
